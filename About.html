Description:
On the home page, there are two buttons; one is for the page's admin and the other is for the student.

Whether the user is an admin or a student he/she must log in first. When the admin logs in successfully he/she has two options; the first option is to register a new student's data (username, ID&password) and the second one is to add a student's marks.

Note: After being logged in, the user can log out from the navigation bar at the top.
On the other hand, the student logs in with the username and password which he/she recieved from the faculty to see his/her marks when updated.
